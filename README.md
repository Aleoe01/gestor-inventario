# Gestor de Inventario – Aplicación de Escritorio en C# (.NET)

Sistema de gestión de inventario desarrollado en C# utilizando .NET y SQL Server, enfocado en la administración de artículos mediante operaciones CRUD y filtros avanzados.

---

## 📌 Descripción

Gestor de Inventario es una aplicación de escritorio construida como proyecto integrador para consolidar conocimientos en desarrollo backend con .NET.

Permite administrar artículos almacenados en una base de datos, aplicando operaciones de alta, modificación, eliminación y búsqueda con distintos criterios.

El proyecto está estructurado en capas para mantener una separación clara de responsabilidades.

---

## 🚀 Funcionalidades

- Listado completo de artículos
- Alta de nuevos registros
- Modificación de artículos existentes
- Eliminación lógica/física con confirmación
- Búsqueda simple y filtros avanzados
- Validaciones de datos
- Manejo de imágenes asociadas a productos
- Control de errores y manejo de excepciones

---

## 🛠 Tecnologías utilizadas

- C#
- .NET (Windows Forms)
- ADO.NET
- SQL Server
- Git / GitHub

---

## 🧠 Arquitectura y conceptos aplicados

- Programación Orientada a Objetos
- Arquitectura en capas (Presentación, Negocio, Datos)
- Separación de responsabilidades
- Conexión y consultas a base de datos con ADO.NET
- Manejo de `SqlConnection`, `SqlCommand` y `SqlDataReader`
- Uso de clases de dominio
- Manejo de excepciones y validación de entradas
- Organización modular del código
